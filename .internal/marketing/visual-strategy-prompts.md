# Visual Strategy & Roadmap Prompts

**Confidential. Image prompts for presenting strategy and execution roadmap.**

---

## Overview

These 18 image prompts create a visual narrative for:
1. **Strategy** (7 images) — Why this opportunity matters
2. **Execution roadmap** (11 images) — How we'll build it

Use these with image generation tools (DALL-E, Midjourney, etc.) or commission custom illustrations.

## Implementation Guide

### How to Use These Prompts

1. **For DALL-E / Midjourney**:
   - Copy prompt as-is into the image generator
   - Adjust dimensions/style as needed
   - Generate 2-3 variations per prompt
   - Select best one and refine

2. **For Custom Illustrations**:
   - Share prompt with designer
   - Provide reference images (style inspiration)
   - Review drafts with team
   - Iterate on design

3. **For Video/Animation**:
   - Use images as storyboard frames
   - Add motion graphics
   - Narrate with strategy message
