# ADR: Introduce Codex CLI plugin support and marketplace integration

**Date**: 2026-04-10
**Status**: Accepted
**Affects**: .codex-plugin/, .claude-plugin/, CHANGELOG.md, README.md, ROADMAP.md, pyproject.toml, skills/, src/llm_router/cli.py, src/llm_router/hooks/codex-post-tool.py
**Risk**: high
**Reversibility**: low

## Decision

To integrate llm-router as a first-class plugin within the Codex ecosystem, making it installable from the Codex marketplace. This expands the reach and usability of the LLM router to Codex users. It also enables specific functionality like post-tool hooks for Codex.
