## Strategic Signals

### 1. Visibility Crisis (Opportunity)

**Signal:** claude-hud (19k stars) became a top-3 plugin by solving ONE problem: showing what Claude is doing.

**Our Gap:** Routing decisions are completely invisible. Users don't know:
- Which model handled their prompt
- Why that model was chosen
- How much it saved them

**v6.0 Action:** Live routing HUD + session replay = address #1 plugin driver.

---

### 2. Memory as Retention Driver (Confirmed)

**Signal:** claude-mem (59k stars, #1 plugin) proves memory/personalization = stickiness.

**Market Fact:** 40% of claude-mem users say it's "worth the plugin slot alone." Cross-plugin memory is valuable.

**Our Opportunity:** Personal routing profiles (v6.1). Users won't switch if router learns their preferences.

---

### 3. Quality = Gatekeeping for Adoption

**Signal:** UncommonRoute's messaging "82% cost savings, 79.4% accuracy, 93.4% pass rate" converts skeptics.

**Barrier:** "Will cheaper model break my code?" is the #1 objection to adoption.

**v6.2 Action:** Published quality benchmarks + quality guard. Answer: *"Haiku is safe for code 94% of the time. We auto-escalate the 6%."*

---

### 4. Ollama Explosive Growth

**Signal:** Ollama GitHub shows:
- 2.9k stars added in 2026-Q1 alone
- **40% user base growth in 3 months**
- Explosive demand for local inference

**Implication:** Free local models are becoming competitive with paid APIs. Our "Ollama-first" strategy is perfectly timed.

**v6.3 Action:** Ollama dashboard + auto-model recommendation. Become the **best Ollama controller**.

---

### 5. Social Proof = Word-of-Mouth Distribution

**Signal:** Plugin marketplace success (1.9k stars on marketplace plugin) = discoverability matters.

**But actual adoption:** Word-of-mouth plugins (claude-mem, claude-hud) grew **10x faster** than marketplace-listed plugins.

**Implication:** Savings card + README badge + leaderboard (v6.4) = user-generated viral distribution.
